Config Sheet - Required Settings

1. Output Folder ID
  • The Google Drive folder ID where the final merged PDF will be saved
  • Find this in the folder URL: drive.google.com/drive/folders/[FOLDER_ID]
  • Copy just the folder ID portion from the URL
2. PDF Name
  • The name for your final merged PDF file
  • Example: 2024_WEP_Combined
  • Do not include the .pdf extension - it will be added automatically

DocIDs Sheet - Document List

  • Column A: Student Name (or document description)
  • Column B: Google Doc ID
    • Find this in the doc URL: docs.google.com/document/d/[DOC_ID]/edit
    • Copy just the document ID portion from the URL
  • Column C: Status (automatically filled during processing)
  • Column D: Error Message (if any errors occur)

How to Use

1 Fill in the Config sheet with Output Folder ID and PDF Name
2 Add all document IDs to the DocIDs sheet (one per row)
3 Click "Create & Merge PDFs" from the WEP Combiner menu
4 Wait for completion - Progress will be shown in the Config sheet and DocIDs sheet
5 Download your PDF - Link will appear in Config sheet when complete
